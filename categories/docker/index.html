Job for docker.service failed

  1. /etc/docker/daemon.json 에서 엔진의 소켓 값을 설정할 수 있음.
  2. /lib/systemd/system/docker.service 에서 엔진의 소켓 값을 설정할 수 있음.
  3. 위의 두 설정 값이 겹치게 되면 도커 엔진을 실행할 수 없게 됨.
